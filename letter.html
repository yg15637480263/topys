如果你有好看的信封，不如给我写信吧
撰文
摇摇冻 2017-03-23 16:56:33
0 0 0

文/摇摇冻@TOPYS

你大概早就忘了上一次提笔写信是怎样的心情了。

信纸一定是要带有淡淡香味的，信封上得点缀些小碎花，粘信封的贴纸也得精致好看。如此折腾一番，再煞有介事地写下隐忍许久的少年心事，如此才对得起那份山长水远来到你身边的仪式感。

Ins上有位叫 Lucy Halcomb 布鲁克林姑娘，自己本身就是位网页设计师，曾替BOBBI BROWN、LAMER等大牌做过官网视觉设计。信息时代，这位妞的爱好倒是独特，偏爱写纸质信，还自己顺手绘制了水彩画信封。

比起设计师，她更喜欢自称为大家的“笔友”。 翻看Lucy的Ins，里面记录的全部全部全部都是她寄出的信笺。时至今日，Lucy的彩绘信封已更新了300多张，灵感来源大多是有趣的日常琐事。有时候Lucy也会把一些流行或复古文化因素融入其中，细致描绘微小细节。

△感恩节特供 网友们在评论区一片哭天抢地“真是爱死了这张邮票”（都给我把关注点给摆正了！）

△告诉全世界，这个周末我去参加party了

△分享学校午餐，金鱼饼干和波伦亚大红肠三明治

△朋友出版了一本关于女人的书，为了帮她庆祝，就挑了几个书里的女人画在信封上

△回扭腰了，玩得开心

△Talkboy 上世纪90年代风靡一时的一款随身听

△为希望干杯

年少时苦恼于辗转小店精心挑选漂亮信封却不得的小纠结，如今即便觅得了这些，你还保有当初那份提笔写信的冲动吗？
